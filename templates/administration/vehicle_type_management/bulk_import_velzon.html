Format du Fichier CSV

Le fichier CSV doit contenir les colonnes suivantes :

  • Nom (requis) - Nom du type de véhicule
  • Description (optionnel) - Description du type
  • Actif (optionnel) - "Oui" ou "Non" (défaut: Oui)
  • Ordre d'affichage (optionnel) - Nombre entier (défaut: 0)
Exemple de ligne CSV :
Voiture,Véhicule de tourisme,Oui,1
Instructions Importantes

  • Les nouveaux types seront créés
  • Utilisez l'encodage UTF-8
  • Vérifiez vos données avant l'import
